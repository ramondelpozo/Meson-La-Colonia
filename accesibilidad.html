1. COMPROMISO DE ACCESIBILIDAD

En Mesón La Colonia estamos comprometidos a garantizar que nuestro sitio web sea accesible para todas las personas, independientemente de sus capacidades técnicas o físicas. Nos esforzamos por cumplir con las pautas de accesibilidad para el contenido web (WCAG 2.1) nivel AA.

2. MEDIDAS DE ACCESIBILIDAD IMPLEMENTADAS

Nuestro sitio web incorpora las siguientes medidas de accesibilidad:

  • Estructura semántica: Uso correcto de encabezados (H1, H2, H3) para facilitar la navegación
  • Contraste de colores: Combinación de colores con suficiente contraste para una lectura clara
  • Texto alternativo: Imágenes con descripciones alternativas para lectores de pantalla
  • Navegación por teclado: El sitio puede ser navegado completamente usando solo el teclado
  • Textos legibles: Tipografía clara y tamaño de texto ajustable
  • Formularios accesibles: Etiquetas claras en todos los campos del formulario de reservas
  • Enlaces descriptivos: Textos de enlace que describen claramente su destino

3. COMPATIBILIDAD CON TECNOLOGÍAS DE ASISTENCIA

Nuestro sitio web es compatible con:

  • Lectores de pantalla (JAWS, NVDA, VoiceOver)
  • Ampliadores de pantalla
  • Software de reconocimiento de voz
  • Navegación mediante teclado

4. LIMITACIONES CONOCIDAS

A pesar de nuestros esfuerzos por garantizar la accesibilidad en todo el sitio web, es posible que haya algún contenido que no sea totalmente accesible. Estamos trabajando continuamente para mejorar la accesibilidad de nuestro sitio.
